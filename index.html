Iris Modal Library

Powerful and flexible library for creating Bootstrap 5 modal dialogs with advanced features

v1.0.1 2025-02-14 Igor Lovriƒá MIT License
Fast & Easy

Simple API for creating dialogs in one line of code

Customizable

Different sizes, types and styles of dialogs

Multi-language

Support for multiple languages with easily extensible translations

Basic Examples

Alert Dialog

Simple informative message with OK button.

Iris.alert('This is an informative message!');
Confirm Dialog

Confirmation with Yes/No buttons that returns a Promise.

Iris.confirm('Are you sure?') .then(result => { console.log(result); // true or false });
Success Dialog

Display a success message.

Iris.success('Operation completed successfully!');
Error Dialog

Display an error with red style.

Iris.error('An error occurred during processing!');
Prompt Dialog

Text input from user with Promise result.

Iris.prompt('Enter your name:', 'Name', 'User') .then(value => { if (value) alert('Entered: ' + value); });
Warning Dialog

Warning with yellow style.

Iris.warning('Attention! This action cannot be undone.');

Dialog Sizes

Different Modal Sizes

Iris supports different dialog sizes using constant values.

// Small size Iris.show({ title: 'Small dialog', size: Iris.SIZE_SMALL }); // Normal size (default) Iris.show({ title: 'Normal dialog', size: Iris.SIZE_NORMAL }); // Large size Iris.show({ title: 'Large dialog', size: Iris.SIZE_LARGE }); // Extra large size Iris.show({ title: 'XL dialog', size: Iris.SIZE_XLARGE }); // Full width Iris.show({ title: 'Full width', size: Iris.SIZE_FULLWIDTH }); // Fullscreen Iris.show({ title: 'Fullscreen', size: Iris.SIZE_FULLSCREEN });

Header Styles

Different Types (Header Colors)

Style the dialog header with Bootstrap colors.

Iris.show({ title: 'Primary style', type: Iris.TYPE_PRIMARY, message: 'The header is blue' });

Advanced Features

Custom Buttons

Create dialogs with custom buttons and actions.

Iris.show({ title: 'Choose Action', message: 'What would you like to do?', buttons: [ { label: 'Save', cssClass: 'btn-success', icon: 'bi bi-save', action: (dialog) => { console.log('Saved!'); } } ] });
Draggable Dialog

Allow user to move dialog using drag & drop.

Iris.show({ title: 'Draggable Dialog', message: 'Grab the header and move me!', draggable: true });
Centered Dialog

Display dialog vertically centered on screen.

Iris.show({ title: 'Centered Dialog', message: 'This dialog is vertically centered', centered: true });
AJAX Loading

Load dialog content from remote server.

Iris.show({ title: 'Loading from Server', ajaxUrl: 'content.html', onContentLoaded: (dialog) => { console.log('Content loaded!'); } });
Form in Dialog

Creating a form with validation inside a dialog.

const dialog = new Iris(); dialog.show({ title: 'Registration', message: '
...
', buttons: [{ label: 'Submit', action: (d) => { /* validate */ } }] });
Callback Functions

React to open, close and other dialog events.

Iris.show({ onshow: (d) => console.log('Before show'), onshown: (d) => console.log('Shown!'), onhide: (d) => { return true; }, onhidden: (d) => console.log('Closed!') });

Dynamic Content

Update Content

Dynamically change dialog content, title and buttons.

const dialog = new Iris(); dialog.show({ title: 'Original' }); dialog.setContent('New content!'); dialog.setTitle('New title'); dialog.addButton({ label: 'New' });
Progress Bar

Display progress inside the dialog.

const dialog = new Iris(); dialog.show({ title: 'Loading...', message: '
...
' }); // Update progress bar dynamically

Multi-language Support

Language Switching

Iris supports multi-language. You can easily add new languages and switch between them.

// Add new language Iris.i18n['de-DE'] = { name: 'Deutsch', translations: { ok: 'OK', cancel: 'Abbrechen' } }; // Set language Iris.setLanguage('de-DE'); // Use it Iris.alert('Hallo Welt!');

Installation and Usage

How to Use Iris?
1. Include Bootstrap 5
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet"> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
2. Include Iris Files
<link rel="stylesheet" href="iris.css"> <script src="iris.js"></script>
3. Use Iris
// Simple Iris.alert('Hello World!'); // Advanced const dialog = new Iris(); dialog.show({ title: 'My Dialog', message: 'Dialog content', size: Iris.SIZE_LARGE, type: Iris.TYPE_PRIMARY, buttons: [{ label: 'OK', cssClass: 'btn-primary' }] });
